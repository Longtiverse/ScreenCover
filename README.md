# ScreenCover

一款轻量级的 Windows 黑屏工具，支持软件黑屏和硬件断电两种模式，可自定义热键，支持输入锁定功能。

## 功能特性

- 🖥️ **软件黑屏** - 全屏黑色窗口覆盖屏幕
- ⚡ **硬件断电** - 使用系统电源管理关闭显示器
- 🔒 **输入锁定** - 拦截鼠标键盘，防止误触
- ⌨️ **自定义热键** - 可自由设置黑屏和锁定热键
- 🐕 **托盘图标** - 笑脸/不笑小狗图标显示当前模式

## 使用方法

### 热键操作

| 功能 | 默认热键 | 说明 |
|------|----------|------|
| 黑屏 | `Win+Shift+H` | 切换黑屏开关 |
| 锁定 | `Win+Shift+L` | 切换输入锁定 |

### 托盘操作

- **单击托盘图标** - 切换黑屏
- **右键托盘图标** - 显示菜单

### 右键菜单

```
黑屏热键: Win+Shift+H   ← 点击修改
锁定热键: Win+Shift+L   ← 点击修改
─────────────────────
切换到: 硬件断电
锁定输入
─────────────────────
退出
```

## 功能说明

### 软件黑屏模式

- 创建全屏黑色窗口覆盖屏幕
- 按任意键退出黑屏
- 鼠标点击无响应，保持黑屏

### 硬件断电模式

- 使用 `SC_MONITORPOWER` 系统调用关闭显示器
- 与系统休眠相同，不会损坏硬件
- 移动鼠标或按键唤醒

### 输入锁定

- 启用后拦截所有鼠标和键盘输入
- 黑屏热键和锁定热键仍然有效
- 再次按锁定热键解除

## 编译

需要 MinGW-w64 或其他 C++ 编译器：

```bash
g++ ScreenCover_simple.cpp -o ScreenCover.exe -lgdi32 -luser32 -lshell32 -mwindows -DUNICODE -D_UNICODE -municode
```

## 配置

热键配置自动保存在注册表：
```
HKEY_CURRENT_USER\Software\ScreenCover
```

## 系统要求

- Windows 10/11
- 无需安装，单文件运行
